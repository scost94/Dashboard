# NGT

Dashboard for data quality analysis and visualization. Tasks done in this project are : 

- Data exploration.
- Three data quality checks.
- Correlation Matrix

# Project structure

* Front End : Angular 7
* Back end : Python
* Database : Firebase


# Instructions

- ```git  clone https://github.com/scost94/NGT ```

- ``` npm install ```

- ``` ng serve --open ```
